# To-Do List

Uma aplicação simples de lista de tarefas (To-Do List) desenvolvida em **JavaScript**, HTML e CSS. Este projeto permite adicionar, marcar como concluídas e remover tarefas, ajudando a organizar tarefas diárias de forma prática.

## Funcionalidades

- Adicionar novas tarefas.
- Marcar tarefas como concluídas.
- Remover tarefas individualmente.
- Limpar todas as tarefas concluídas.
- Armazenamento local (LocalStorage) para manter as tarefas mesmo após atualizar a página. *(se implementado)*

## Tecnologias

- **JavaScript** – lógica da aplicação.
- **HTML5** – estrutura da página.
- **CSS3** – estilização e layout.
